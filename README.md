# LXE AGENT

> **Project:** LXE AGENT
> **Version:** v0.1 - beta

## Introduction

本项目是一个基于 Python 的 agent 机器人，专门用于 FBA 业务模块。


## Quick Install

```powershell
irm https://raw.githubusercontent.com/LXE123/LXE_AGENT_LOCAL_FBA/main/scripts/install.ps1 | iex
```

安装完成后使用：

```powershell
LXEFBA start
```
