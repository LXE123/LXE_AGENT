# Python 3.12.10 / uv 部署说明

本项目统一使用 uv 管理 Python 版本、虚拟环境和依赖锁定。

## 一键安装

在项目根目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\install.ps1
```

脚本会自动完成：

- 安装或发现 uv
- 安装 Python 3.12.10
- 按 `uv.lock` 同步 `.venv`
- 安装 Playwright Chromium
- 执行基础 import 检查

## 手工命令

```powershell
uv python install 3.12.10
uv sync --frozen --all-groups --python 3.12.10
uv run --frozen python -m playwright install chromium
uv run --frozen python -c "import sys; print(sys.version)"
```

启动主入口：

```powershell
uv run --frozen python main.py
```

## 依赖管理规则

- `pyproject.toml` 是直接依赖声明入口。
- `uv.lock` 是部署时的依赖真相来源。
- 不再维护 `requirements.txt`。
- 添加运行依赖：`uv add <package>`
- 添加测试/开发依赖：`uv add --dev <package>`
- 更新锁文件：`uv lock`
