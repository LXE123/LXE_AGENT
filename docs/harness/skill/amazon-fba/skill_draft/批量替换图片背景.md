批量替换图片背景
---
用户发送一个压缩包，里面都是图片，同时用户也会说明需要给这个压缩包里面的图片批量替换什么样的背景
例子：
把这个压缩包里的所有图片背景替换成沙漠风格
---
步骤
第一步：解析压缩包，把里面的图片提取出来
第二步：把这些图片批量发送到 LLM 图生图 API 接口
第三步：把生成好的图片的url地址发送给用户
完成
---

curl -X POST "https://open.bigmodel.cn/api/paas/v4/images/generations" \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
        "model": "glm-image",
        "prompt": "修改背景，沙漠风格",
        "size": "1280x1280"
    }'
