先看第一步
用户会发送一个 excel 文件，
（命名类似 "4.26-SP260414001-新棱镜备货-美国（4.28）-2.xlsx"，格式大概为“日期-SP单号-其它信息-国家（又是一个日期）”，注意，重要的是中间的单号和国家，在 SKILL 中要求 AI 自己从文件名中提取这两个信息，如果拿不到就和用户说缺少关键信息）
然后要求执行填写报关单流程。

这个 xlsx 文件中需要的数据在第三个 sheet。
这第三个 sheet 第一行一定是下面这样的
“日期	SKU	品名	规格型号	发货量	单价	供货商	采购订单号	采购总价	商品名称	售价	总价	单位”
具体需要脚本提取的列有这些：
品名    规格型号    发货量  商品名称    售价    总价    单位
不过具体提取几行还是取决于第一列有几行有效数据。


进入第二步（这个第二步分成几个小步骤）
第二步第一阶段：
data\customs_declaration\custom_declaration_documents.xlsx
这个文件是报关资料的填写模板
目前第一步是在这个 excel 文件中找到命名为 `申报要素` 的 sheet。
在这个 sheet 中找到这一列：
序号	品名	海关HS编码	检疫附加码	品牌	型号	品牌类型	出口享惠情况	其他申报要素

接下来是怎么填写：
1. `序号`这一列从 1 开始顺序填写数字
2. `品牌`这一列都固定填写：“0无品牌”	
3. `品牌类型`这一列都固定填写：“0无品牌”
4. `出口享惠情况`这一列都固定填写：“0不享惠”
5. 第一步拿到的 `规格型号` 写入 `型号` 这一列
6. 第一步拿到的 `商品名称` 写入 `品名` 这一列
接下来是有条件的填写
7. 根据第一步中的数据做判断：
7.1 表带类：
- 如果`商品名称`中存在关键字`表带`，查看`品名`是否包含关键字`米兰尼斯`或`米兰`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113200000`，`其他申报要素`这一列写入`贱金属`。
- 如果`商品名称`中存在关键字`表带`，查看`品名`是否包含关键字 `编织` 或 `尼龙`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113900090`，`其他申报要素`这一列写入`尼龙`。
- 如果`商品名称`中存在关键字`表带`，查看`品名`是否包含关键字 `真皮` 或 `尖尾皮带`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113900090`，`其他申报要素` 这一列写入`皮革`。
- 如果`商品名称`中存在关键字`表带`，查看`品名`是否包含关键字 `硅胶`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113900090`，`其他申报要素` 这一列写入`硅胶`。
以下类型的表带可以不根据品名判断
- 如果`商品名称`中存在关键字`编织表带`。在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113900090`，`其他申报要素` 这一列写入`尼龙`。
- 如果`商品名称`中存在关键字`尼龙表带`。在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9113900090`，`其他申报要素` 这一列写入`尼龙`。

7.2 表壳类
- 如果`商品名称`中存在关键字`表壳`，查看`品名`是否包含关键字 `金属`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9111800000`，`其他申报要素` 这一列写入`贱金属`。
- 如果`商品名称`中存在关键字`表壳`，查看`品名`是否包含关键字 `PC`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9111800000`，`其他申报要素` 这一列写入`PC`。
- 如果`商品名称`中存在关键字`表壳`，查看`品名`是否包含关键字 `TPU`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9111800000`，`其他申报要素` 这一列写入`TPU`。
- 如果`商品名称`中存在关键字`表壳`，但是`品名`不包含上面列出的关键字。在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `9111800000`，`其他申报要素` 这一列写入`PC`。

7.3 包装盒类
- 如果`商品名称`中存在关键字`包装盒`。在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `4819200000`，`其他申报要素` 这一列写入`纸质+塑料`。

7.4 手表保护套类
- 如果`商品名称`中存在关键字`手表保护套`，查看`品名`是否包含关键字 `PC`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `3926909090`，`其他申报要素` 这一列写入`PC`。
- 如果`商品名称`中存在关键字`手表保护套`，查看`品名`是否包含关键字 `TPU`。如果有，在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `3926909090`，`其他申报要素` 这一列写入`TPU`。
- 如果`商品名称`中存在关键字`手表保护套`，但是`品名`不包含上面的关键字。在第二阶段`申报要素` sheet 中的 `海关HS编码` 这一列写入 `3926909090`，`其他申报要素` 这一列写入`PC`。

7.5 其它
- 如果上面列出的情况都不符合，就在`海关HS编码`和`其他申报要素`留空，并且写入脚本返回内容里的 `notice` 字段

下面是第二步第二阶段
查看 custom_declaration_documents.xlsx 的 `报关单` sheet
找到这一列
```xlsx
项号	商品编号		商品名称及规格型号			数量及单位		单价/总价/币制		原产国（地区）		最终目的国（地区）			境内货源地			征免	净重	毛重	件数
```
做下面操作
1. 这一列中找到 `单价/总价/币制` 这个合并单元格，我们要输入的内容是`单价`（从第一阶段中的`售价`这一列获取），先写下面一个输入第一个，然后每 3 行输入一个。
2. 这一列中找到 `数量及单位` 这个合并单元格，这个单元格下面有两列，第一列我们要输入的内容是 `数量`（从第一阶段中的`发货量`这一列获取），第二列我们要输入的内容是 `单位`（从第一阶段中的`单位`这一列获取）。
3. 现在就是文件名中的 sp 单号和国家发挥作用的时候了。
3.1 目前 agent 已经有一个 skill 会根据 sp 单号下载`装箱数据`，这个`装箱数据`会保存在本地，`装箱数据`中有重要的信息：`毛重`（装箱数据中只会有一个 sheet，查看第一行，找到毛重这一列提取出所有的装箱的 `毛重` 加在一起）。然后根据刚才填写的 `数量` 按比例分配这些 `毛重`（最多一位小数），填写进 `报关单` sheet 的 `毛重` 这一列 。(如果分配好的重量加在一起不等于装箱数据里的毛重，就挑选出最大重量的那个商品修改 `毛重`)
3.1.1 `报关单` sheet 的 `净重` 这一列就等于 `毛重` 这一列的每个商品的重量减 0.1 或 0.01（防止为0，如果减 0.1 等于 0, 就减 0.01，如果减 0.01 等于 0，那就不管了）。
3.2 然后`装箱数据` 中的 `箱序号` 这一列的最大值就是箱数，填写进 `报关单` sheet 的 `件数` 这一列（只用填写 `件数` 下的第一个单元格）。
3.3 从文件名中提取出的国家就是目的地，填写进 `报关单` sheet 的 `最终目的国（地区）` 这一列
